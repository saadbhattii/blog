# blog

Hi, my name is Saad Bhatti and this is my blog. I learnt recently that the art of conveying a message is where the artist disappears, and the message remains. Hence, by prioritizing simplicity, I have decided that this repo is my blog, in which each folder will be dated with the title, and will contain my specific blog post as a README.md file. 

I will be talking about quantum computing, computer science, mathematics, statistics or anything I find interesting as I explore further. Feel free to create a discussion to point something out. The goal is to express, get corrected, and learn further. 

Cheers.
